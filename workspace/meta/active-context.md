# Active Context

Use this file for short-lived continuation state: "stop today, continue later".
Keep only the last 1-2 weeks of useful context, about 20 lines maximum.

## Directory

## Use

Read this file only when resuming, pausing, or checking current open threads.

## Read Path From Here

1. Start from the newest relevant line in [Recent Continuations](#recent-continuations).
2. Open only the file paths named in that line.
3. If no line matches, return to `AGENTS.md` and use its task read paths.
