## Examples

1. User asks for a domain SOP. Agent reads `RUNTIME.md`, then the matched SOP branch, then answers with a short checklist.
2. User asks to update domain memory. Agent writes a candidate first unless the user explicitly requests direct edit.
3. User asks for risky action. Agent applies global permission tiers and asks before P3 operations.
