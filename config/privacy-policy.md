# Privacy Policy Template

## Directory

- [Local First](#local-first)

## Local First

Iron Agent stores workspace files locally in the installed folder.

## Model Calls

Codex may read files you explicitly route into a task. Optional Codex automation
can call a model on a schedule after user approval.

## Secrets

Do not store secrets in Iron Agent files. Use environment variables or ignored
local config files.
